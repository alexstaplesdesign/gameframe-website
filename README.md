# GameFrame

Website for GameFrame — a custom sports graphics service for young athletes.
Customers submit photos and get back posters, LED signs, and baseball
card-style graphics.

## Stack

HTML5 · Tailwind CSS (CDN) · JavaScript (vanilla)

## Running it

Open `index.html` in a browser or use Live Server in VS Code.

## Pages

- `index.html` — Landing page with samples, pricing, and FAQ
- `portfolio.html` — Full portfolio gallery

## Adding samples

Drop new images into `media/` and add the filename to the `allSamples`
array in both `index.html` and `portfolio.html`. The newest five show
on the main page automatically; all of them show on the portfolio page.
